## 特殊符号

| HTML源代码 | 显示结果 | 描述                   |
| ---------- | -------- | ---------------------- |
| `&lt;`     | &lt;     | 小于号或显示标记       |
| `&gt;`     | &gt;     | 大于号或显示标记       |
| `&amp;`    | &amp;    | 可用于显示其他特殊字符 |
| `&quot;`   | &quot;   | 引号                   |
| `&reg;`    | &reg;    | 已注册                 |
| `&copy;`   | &copy;   | 版权                   |
| `&trade;`  | &trade;  | 商业                   |
| `&nbsp;`   | &nbsp;   |                        |
